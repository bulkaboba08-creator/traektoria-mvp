# Траектория — MVP цифровой платформы сквозного кадрового аудита

Демонстрационный прототип (MVP) для заявки на конкурс **Росмолодёжь.Гранты**.

**Живая версия:** https://bulkaboba08-creator.github.io/traektoria-mvp/

Концепция платформы: Бакиров Айдар Ильдусович. Свидетельство о депонировании
объекта интеллектуальной собственности **№4405866 от 30.06.2026**.

---

## Что это

«Траектория» — концепция единой государственной платформы, которая собирает
*верифицированный* цифровой профиль компетенций человека (школа → вуз → карьера)
из государственных источников и даёт работодателям объективный автоматический
скоринг кандидатов вместо «резюме вслепую».

Этот репозиторий — **рабочий прототип интерфейса**, показывающий три роли:

| Раздел | Что демонстрирует |
|---|---|
| **Гражданину** (`citizen.html`) | Профиль-траектория с бейджами верификации (Рособрнадзор, ФРДО, СФР), радар компетенций, сводный индекс, выдача временного кода доступа работодателю (контур ФЗ‑152) |
| **Работодателю** (`employer.html`) | Фильтры вакансии → **работающий алгоритм скоринга** по базе из 24 кандидатов → ранжированный топ‑лист с детализацией баллов |
| **Ведомству** (`analytics.html`) | Дашборд кадрового потенциала региона: дефицит по отраслям, рост профилей, доля северного стажа и ДПО, трудоустройство по вузам |
| **Планирование** (`strategic.html`) | Панель ведомства по методологии нацпроекта «Кадры» (7-летний горизонт, до 2033): **интерактивная карта кадрового дисбаланса** по 63 районам РБ с фильтром отраслей (индекс дефицита red/yellow/green), прогноз потребности с зонами (тактика/демография/стратегия) и ползунком роботизации, автоматические директивы по КЦП |

## Важно

- Все персональные данные в прототипе **вымышленные**. Это демонстрация интерфейса и логики.
- Полноценная система требует государственной интеграции (ЕСИА, реестры Рособрнадзора,
  ФРДО, электронная трудовая книжка СФР) — она недоступна на этапе прототипа.

## Технически

Статический сайт: чистые HTML5 + CSS + JavaScript, без сборщиков и внешних зависимостей.
Открывается как по ссылке, так и локально двойным кликом по `index.html`.

```
index.html        — главная (выбор роли)
citizen.html      — кабинет гражданина
employer.html     — кабинет работодателя (скоринг)
analytics.html    — аналитика региона
strategic.html    — стратегическое планирование 2030–2055 (тепловая карта + прогноз)
css/styles.css    — дизайн-система (стиль Госуслуг)
js/data.js        — демонстрационные данные (кандидаты, вакансии, статистика, прогноз)
js/shared.js      — общий каркас + алгоритм скоринга
js/bashmap.js     — границы районов Башкортостана (OSM, упрощены) для карты
js/govdata.js     — данные по районам (вакансии/выпускники/индекс) и прогноз потребности
assets/logo.svg   — логотип
```
